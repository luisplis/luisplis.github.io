Web Corporativa en **Wordpress** para la empresa Web Financial Group con **diseño responsive** y creación de **tema 
a medida** con acentos visuales.

La web incluía los apartados de noticias, información corporativa, ofertas de empleo, casos de éxito y 
servicios.

Gestión de diferentes **custom post types** con maquetación específica para los casos de éxito y las ofertas de empleo.

Los casos de éxito se mostraban en forma de **sliders reutilizables** en diferentes secciones de la web para mejorar
la experiencia de usuario.

El **posicionamiento orgánico** fue mejorado con la optimización de imágenes y el uso mínimo de recursos gracias al
tema a medida obteniendo un excelente **rendimiento** en el test Web Core Vitals de Google.
