La forma tradicional de mostrar información financiera siempre se ha realizado con tablas, gráficos y datos, primando
siempre la información sobre la estética. En muchos casos aún se pueden ver portales con una maquetación básica 
que arroja datos sin una categorización que cuesta entender de un primer vistazo.

La demo es una propuesta realizada para mejorar en muchos sentidos la información financiera mostrada, manteniendo 
la esencia y adecuando la información a la forma actual de consumir datos y trabajar con ellos.

Para esta demostración se generaron dos propuestas en HTML, CSS y JS junto con un vídeo 
para demostrar que el uso de las nuevas tecnologías puede hacer evolucionar la forma de mostrar, 
consumir y difundir información financiera.

La web demo es una propuesta más visual, sin perder el enfoque tradicional. Con esta idea se mejora
la experiencia de usuario al categorizar la información, hacerla accesible en cualquier dispositivo y 
explorar la forma de compartir dicha información en todos los canales posibles.

Como dato interesante, se contempló la 
<a href="https://shotstack.io/" title="generación de vídeo mediante API y Json" class="link" target="_blank" rel="noopener noreferer">generación de vídeos programados al vuelo</a> 
con los datos obtenidos por API conformando una forma puntera de compartir información financiera más atractiva.
